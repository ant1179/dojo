# AGENTS.md

## Rôle de Codex
Codex est responsable de l’implémentation complète des tâches issues du backlog.

## Rôle humain
L’humain définit les specs, valide les PR et prend les décisions produit.

## Règles
- Toute tâche doit être liée à une issue
- Toujours écrire des tests
- Toujours lancer la CI avant PR
- Ne jamais merger directement
- PR obligatoire pour toute modification

## Workflow
Issue → implémentation Codex → PR → CI → validation humaine → merge
